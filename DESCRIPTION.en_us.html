Calistoga was inspired by lettering seen on the posters Oscar M. Bryn made for the Santa Fe Railroad.

Calitoga is a cheerful display face whose style can be associated with Railroads and the West of the United States.

The Github repository for this project is here: https://github.com/SorkinType/Calistoga

Calistoga is an original typeface designed by Yvonne Schuttler

Eben Sorkin expanded the language support and refined the design in 2018.

Calistoga is published by Sorkin Type.
